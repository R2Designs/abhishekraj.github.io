Instant Refunds

Our users used to face problem due to the time it takes to transfer big money, as the refund used to take 3-7 days we wanted to help our users in this scenario. we built a way so that as soon as the user asks for a refund, we planned to provide the refund from pagarbook's end which will be a lot faster to transfer and we can equate this with refund which we should recieve.

Pagarbook Loans

We wanted to bring loans to our credit worthy users, so we implemented a solution where on approval from the employer, the employees are able to borrow loans from pagarbook. The added layer is an added measure for security where we attract or cater to trustworthy employees.
