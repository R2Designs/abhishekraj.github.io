Group Ordering

Task for a recent successful hiring interview

Problem

An interesting data pattern at Swiggy/Zomato suggests that people order food from multiple restaurants to be delivered at the same address. These are larger orders when compared to the ordering pattern of the user from that location. The team has come up with the hypothesis that these orders are made for parties & large gatherings. They are exploring a new feature which allows multiple people to add food to the single order using their own devices.



Understanding Problem

What: Solution for Group Ordering

Who: Building it for a Swiggy user who orders in bulks presumably for functions and parties

Why: Its so that the next time he group orders, he has a nicer and simpler experience

Where: IOS App, ordering where the event takes place

User Story

We believe that Swiggy Group Orders for food buyers will help us increase users and the food orders

We will know we’re successful when our buyer has completed the flows and uses it more times to place more orders

Needs of event host

The host of the event can order for everybody and pay for everybody too

While enjoying the event, Still able to place orders for everybody with ease

The food should arrive on time and should arrive together so that everybody can start eating at the same time

Needs of event attendee

They should be able to order my favourite food from my favourite restaurant

Food should be fresh and should arrive at same time so that they can have great discussions with the team while completing food

They dont want to input my addresses every event they go in

Flows of event host

The host of the event opens group buy

The host names the event

The host shares the qr/code with his friends

The attendees orders from their favourite restaturants

The host gets the notification of new orders from his friends available in super cart

The host orders the food list from super cart

The order is delivered all together



Flows of event attendee

The attendee recieves the qr/code from the host

the attendee picks his favourte restaurant and sends the order to the super cart

The host gets the notification of new order and places the order for the attendee

The order is delivered all together
